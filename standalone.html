Guide d'utilisation - iSchémateur

Par Normand Rocheleau ing.

1. Bibliothèques de composants

  • Créez une nouvelle bibliothèque ou utilisez la bibliothèque par défaut
  • Chaque bibliothèque contient des modèles de composants entraînés
  • Exportez/importez vos bibliothèques au format JSON

2. Entraînement

  • Téléversez un schéma de référence
  • Cliquez sur "Entraîner" pour marquer les composants
  • Dessinez des rectangles autour des composants
  • Nommez chaque type de composant

3. Analyse

  • Téléversez un schéma à analyser
  • Cliquez sur "Analyser" pour détecter les composants
  • Ajustez le seuil de confiance pour filtrer les résultats
  • Zoom avec la molette, pan en glissant

4. Disjoncteurs interactifs

  • Cliquez sur un disjoncteur pour basculer son état
  • Vert = FERMÉ, Rouge = OUVERT
